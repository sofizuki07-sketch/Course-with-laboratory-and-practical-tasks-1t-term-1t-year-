# Лабораторная работа №1
## Тема: Основы Rust

### Задача 1: Система подсчёта оценок студентов
Создайте программу на Rust, которая работает с оценками студентов. Реализуйте:
- Перечисление `Grade` с вариантами оценок: `A`, `B`, `C`, `D`, `F`
- Структуру `Student` с полями: `name` (строка), `grades` (массив из 5 оценок), `average_score` (число с плавающей точкой)
- Функцию `calculate_average`, которая принимает массив оценок и возвращает средний балл
- Функцию `grade_to_points`, которая преобразует оценку из перечисления `Grade` в числовое значение (A=5, B=4, C=3, D=2, F=0) с использованием `match`
- В функции `main` создайте трёх студентов с разными оценками, вычислите их средние баллы, определите лучшего студента и выведите результаты с использованием циклов

### Задача 2: Конвертер валют с обработкой ввода
Разработайте программу-конвертер валют, которая:
- Определяет перечисление `Currency` с вариантами: `USD`, `EUR`, `RUB`, `JPY`
- Хранит курс валют в массиве или структуре (например, 1 USD = 90 RUB, 1 EUR = 100 RUB и т.д.)
- Запрашивает у пользователя: исходную валюту, целевую валюту и сумму для конвертации
- Использует `match` для определения выбранных валют
- Вычисляет результат конвертации и выводит его
- Обрабатывает ошибки ввода (некорректные числа, неизвестные валюты) с выводом понятных сообщений об ошибках
- Предлагает пользователю повторить операцию или завершить программу с использованием цикла

## Теоретическая справка по теме «Основы Rust»

### 1. Основные новые понятия темы:

- **Система типов Rust** – строгая статическая типизация с автоматическим выводом типов
- **Владение и заимствование** – уникальная система управления памятью без сборщика мусора
- **Неизменяемость по умолчанию** – переменные неизменяемы, если явно не указано `mut`
- **Сопоставление с образцом** – мощный оператор `match` для обработки разных вариантов

### 2. Ключевые изученные элементы языка Rust:

- Объявление переменных с `let` и типами (`i32`, `f64`, `String`, `&str`)
- Структуры (`struct`) для создания пользовательских типов данных
- Перечисления (`enum`) для представления вариантов выбора
- Условные операторы (`if`, `else if`, `else`) и циклы (`for`, `while`)
- Функции с явным указанием типов параметров и возвращаемого значения
- Обработка ошибок через `Result` и `match`
- Ввод-вывод через `std::io` и форматированный вывод с `println!`

### Ключевые моменты для запоминания:

- Rust требует явного указания `mut` для изменяемых переменных
- Строки бывают двух типов: `String` (владеющая) и `&str` (заимствованная)
- `match` должен обрабатывать все возможные варианты перечисления
- Массивы имеют фиксированный размер, векторы (`Vec`) – динамический
- Функции возвращают последнее выражение без точки с запятой

### Практические советы:

- Внимательно читайте сообщения компилятора – они помогают исправить ошибки
- Начинайте с простых структур и постепенно добавляйте функциональность
- Тестируйте программу с разными входными данными, включая ошибочные
