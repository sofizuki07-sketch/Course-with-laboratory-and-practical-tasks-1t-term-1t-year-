# Лабораторная работа №1
## Тема: Указатели и динамическое выделение памяти

### Задача 1: Тройной указатель и динамическая память
Создайте указатель типа `int***` и организуйте через три уровня косвенности доступ к целочисленной переменной. Используя только функции динамического выделения памяти (`malloc`), организуйте цепочку указателей так, чтобы конечное значение вычислялось как сумма двух чисел, введённых пользователем с клавиатуры. Реализуйте ввод чисел через промежуточные указатели, а результат выведите через тройной указатель. После вывода результата освободите всю выделенную память в правильном порядке. Не используйте дополнительные переменные кроме указателя `triple_ptr`.

### Задача 2: Поиск минимального элемента массива через указатели
Напишите программу, которая создаёт динамический массив целых чисел размером `n` (вводится с клавиатуры), заполняет его значениями с клавиатуры, а затем находит минимальный элемент массива, используя только арифметику указателей (инкремент/декремент) без применения оператора `[]`. Результат выведите на экран. После выполнения освободите память.

### Задача 3: Подсчёт гласных букв в строке через указатели
Напишите программу, которая считывает строку с клавиатуры (максимальная длина – 255 символов) и подсчитывает количество гласных букв (русских и английских) в ней, используя только указатели для прохода по строке. Программа должна работать как со строчными, так и с заглавными буквами. Результат вывести на экран.

### Задача 4: Функция копирования массива через указатели
Напишите функцию `copyArray`, которая принимает два указателя на `int` и размер массива, и копирует содержимое первого массива во второй, используя только арифметику указателей (без индексации `[]`). В `main` продемонстрируйте работу функции: создайте два динамических массива, заполните первый, скопируйте его во второй и выведите оба массива для проверки.

### Задача 5: 
Создайте программу для работы с динамической матрицей целых чисел, которая позволяет пользователю:
1. Задать начальные размеры матрицы (строки, столбцы)
2. Заполнить матрицу значениями
3. Динамически изменить размер матрицы (увеличить или уменьшить количество строк/столбцов) с сохранением существующих данных там, где это возможно
4. Вывести матрицу до и после изменения размеров

Реализуйте два режима работы с памятью:
- **Режим A**: Разрозненное выделение (отдельные строки)
- **Режим B**: Непрерывное выделение (один блок памяти)

Программа должна автоматически перераспределять память при изменении размеров, корректно копируя старые данные в новую область. Предусмотрите обработку ситуаций, когда новые размеры меньше старых (данные обрезаются) или больше (новые ячейки заполняются нулями).

## Теоретическая справка по теме "Указатели и динамическая память в C"

### Основные понятия:
1. **Указатели** – переменные, хранящие адреса других переменных или областей памяти.
2. **Динамическое выделение памяти** – запрос памяти во время выполнения программы с помощью функций `malloc`, `calloc`, `realloc`.
3. **Освобождение памяти** – возврат выделенной памяти системе с помощью `free`.

### Ключевые моменты для запоминания:
- Указатель объявляется с символом `*`: `int *ptr;`
- Для получения адреса переменной используется оператор `&`: `ptr = &x;`
- Для доступа к значению по адресу используется оператор `*`: `*ptr = 10;`
- Динамическая память выделяется функцией `malloc(size)` и освобождается `free(ptr)`
- Арифметика указателей позволяет перемещаться по массивам без использования индексов
- Указатели на функции позволяют передавать код как параметр

### Практические советы:
- Всегда проверяйте результат `malloc` на `NULL` (неудачное выделение)
- После `free` устанавливайте указатель в `NULL`, чтобы избежать "висячих указателей"
