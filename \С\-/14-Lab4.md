# Лабораторная работа №4
## Тема: Продвинутая работа с функциями и проектами на C

### Задача 1: Реализация стека с функциями работы через указатели
Создайте структуру `Stack`, содержащую динамический массив целых чисел, индекс вершины стека и максимальную ёмкость. Реализуйте функции:
- `createStack` – создание стека заданного размера,
- `push` – добавление элемента (возвращает код ошибки при переполнении),
- `pop` – извлечение элемента (возвращает код ошибки при пустом стеке),
- `isEmpty` и `isFull` – проверки состояния,
- `freeStack` – освобождение памяти.
Продемонстрируйте работу стека, обрабатывая ошибки через возвращаемые коды.

### Задача 2: Функция сортировки с пользовательским компаратором
Напишите функцию `sortArray`, которая принимает массив структур `Book` (название, автор, год), размер массива и указатель на функцию-компаратор. Компаратор должен определять порядок сортировки: по названию, по автору или по году. Реализуйте три разные функции-компаратора. В основной программе создайте массив книг, отсортируйте его разными способами и выведите результаты.

### Задача 3: Функция чтения матрицы из файла с обработкой ошибок
Создайте функцию `readMatrixFromFile`, которая читает матрицу целых чисел из текстового файла. Функция должна:
- принимать имя файла,
- возвращать динамически созданную матрицу через параметры-указатели,
- возвращать код ошибки (файл не найден, неверный формат, недостаточно памяти),
- определять размеры матрицы автоматически по содержимому файла.
Дополнительно напишите функцию `printMatrix` для вывода матрицы на экран.

### Задача 4: Рекурсивный обход каталога с подсчётом файлов
Напишите программу, которая использует рекурсивную функцию для обхода заданного каталога и его подкаталогов. Функция должна подсчитывать:
- общее количество файлов,
- количество файлов с расширением `.txt`,
- суммарный размер всех файлов.
Используйте функции библиотеки `dirent.h` и `stat.h`. Результаты выводите в читаемом формате.

### Задача 5: Функция для работы с комплексными числами
Создайте структуру `Complex` для представления комплексных чисел. Реализуйте функции:
- `addComplex`, `subtractComplex`, `multiplyComplex`, `divideComplex` – арифметические операции,
- `complexToString` – преобразование в строку для вывода,
- `parseComplex` – создание из строки вида "a+bi".
Организуйте код в отдельных файлах: `complex.h`, `complex.c`, `main.c`. В основной программе продемонстрируйте все операции.

## Теоретическая справка по теме "Продвинутая работа с функциями и проектами на C"

### 1. Основные понятия темы:
- **Указатели на функции** – возможность передавать функции как аргументы другим функциям
- **Рекурсивные функции** – функции, вызывающие сами себя
- **Обработка ошибок через возвращаемые коды** – альтернатива исключениям в C
- **Модульная организация кода** – разделение на заголовочные (.h) и исходные (.c) файлы
- **Динамические структуры данных** – структуры, размер которых изменяется во время выполнения программы

### 2. Ключевые изученные элементы языка C:
- Функции с параметрами-указателями для возврата нескольких значений
- Рекурсивные алгоритмы и условия их завершения
- Функции-компараторы для сортировки и поиска
- Организация многофайловых проектов*
- Обработка ошибок через возвращаемые значения и коды ошибок
Предлагаю ознакомится с данными темами, понятиями самостоятельно в Интернет-ресурсах и справочниках

*В файле "Памятка по работе с Makefile" моэете узнать, как установить необходимые материалы для создаания и сборки многофайловых проектов через один из способов - Makefile соответственно.
