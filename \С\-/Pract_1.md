## Часть 1. Установка и настройка среды разработки

### 1.1. Установка компилятора GCC

#### Для Windows (рекомендуется MSYS2 + UCRT64):
1. Скачайте **MSYS2** с официального сайта: [msys2.org](https://www.msys2.org/)
2. Запустите установщик и следуйте инструкциям.
3. После установки откройте **MSYS2 UCRT64** из меню «Пуск».
4. Обновите пакеты, выполнив команды:
   ```bash
   pacman -Syu
   pacman -Su
   ```
5. Установите GCC:
   ```bash
   pacman -S mingw-w64-ucrt-x86_64-gcc
   ```
6. Проверьте установку:
   ```bash
   gcc --version
   ```

#### Для macOS (через Homebrew):
1. Установите **Homebrew** (если ещё не установлен):
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. Установите GCC:
   ```bash
   brew install gcc
   ```
3. Проверьте установку:
   ```bash
   gcc-13 --version  # или другая версия, установленная через brew
   ```

#### Для Linux (Ubuntu/Debian):
```bash
sudo apt update
sudo apt install gcc
gcc --version
```

### 1.2. Установка и настройка VSCode
1. Скачайте VSCode с [официального сайта](https://code.visualstudio.com/).
2. Установите расширения для C/C++:
   - C/C++ Extention Pack (от Microsoft)
   - Code Runner (опционально, для быстрого запуска кода) ИЛИ vs-code-runner
3. Настройте интеграцию с GCC:
   - В Windows: укажите путь к GCC из MSYS2 UCRT64 в `PATH` (подробную инструкцию см. в файле "Настройка переменной PATH для MSYS2 GCC для Windows").
   - В macOS/Linux: GCC обычно уже доступен в системе.
4. Установите поддержку кириллицы при выводе команд на экран (подробную инструкцию см. в файле "Настройка вывода русского языка в VS Code").

## Часть 2. Решение задач на языке C

### Задача 1: Вычисление факториала  
Напишите программу, которая запрашивает у пользователя целое число `n` и вычисляет его факториал. Используйте цикл `for`.  
Проверьте работу программы для `n = 5`, `n = 7`.
**Теория:**  
Факториал числа `n` (обозначается `n!`) — это произведение всех натуральных чисел от 1 до `n`. Например:  
`5! = 1 * 2 * 3 * 4 * 5 = 120`.  
В программировании для вычисления факториала удобно использовать цикл `for`.
Подробнее о цикле см. здесь: https://metanit.com/c/tutorial/2.12.php

### Задача 2: Проверка числа на простоту 
Напишите программу, которая принимает число `n` и определяет, является ли оно простым.  
**Теория:**  
Простое число — это натуральное число больше 1, которое делится без остатка только на 1 и на само себя.  
Подсказка для эффективности кода!: Для проверки простоты числа достаточно проверить делимость на все числа от 2 до `sqrt(n)`.
Подробней об условных операциях см. здесь: https://metanit.com/c/tutorial/2.7.php  об условных конструкциях: https://metanit.com/c/tutorial/2.11.php

### Задача 3: Сумма и произведение цифр числа  
Напишите программу, которая запрашивает у пользователя целое положительное число, вычисляет сумму и произведение его цифр с использованием цикла и операций деления, а затем выводит оба результата на экран. 
**Теория:**  
Для работы с цифрами числа используются операции:
- `%` — остаток от деления (для получения последней цифры).
- `/` — целочисленное деление (для удаления последней цифры).

# Задача 6: Сортировка массива выбором (Selection Sort)
Напишите программу, которая запрашивает у пользователя размер массива n (не более 100 элементов), затем заполняет массив целыми числами с клавиатуры. Реализуйте алгоритм сортировки выбором по возрастанию: на каждом шаге находите минимальный элемент в неотсортированной части массива и меняйте его местами с первым элементом этой части, постепенно увеличивая отсортированную часть. 
В конце программа должна вывести на экран массив до и после сортировки.
**Теория:**  
Алгоритм сортировки выбором — один из простейших алгоритмов сортировки. Его суть в том, что на каждом шаге мы находим минимальный (или максимальный) элемент из неотсортированной части массива и меняем его местами с первым элементом неотсортированной части. Таким образом, отсортированная часть массива растёт слева направо.
Подробней см. здесь: https://habr.com/ru/articles/422085/
